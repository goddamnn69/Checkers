# Changelog


## [1.2.0] - 2026-04-24

### Added
- Кнопка "ЗАНОВО" — перезапускает игру с теми же настройками
- Анимация мелькания фона при смене игрока
- Закрытие сетевого соединения при возврате в лобби

### Changed
- Кнопки "В ЛОББИ" и "ЗАНОВО" расположены в один горизонтальный ряд
- `Player` переделан из статического класса в обычный — каждая игра создаёт свой экземпляр
- `GameEngine.ExecuteMove` и `Bot.MakeBotMove` принимают `Player` параметром
- Подписка на событие смены игрока через ViewModel вместо статического события

### Fixed
- Утечка памяти из-за статического события `PlayerSwitched`
